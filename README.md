# Lock In Fitness - Fitness Website UI/UX Design

This project was created as part of the Future Interns UI/UX Design Internship Program.

## Project Overview

Lock In Fitness is a modern fitness website UI/UX design created to provide users with a clean, engaging, and user-friendly experience while exploring fitness services, membership plans, and free trial registration.

The website allows users to:

- Explore fitness services
- View membership plans
- Register for a free 7-day trial
- Learn about gym equipment and facilities
- Book a virtual fitness session
- Contact the fitness team

## Pages Included

- Home Page
- Membership Page
- Services Page
- Free Trial Registration / Contact Page

## Tools Used

- Figma
- GitHub

## Design Style

- Modern UI
- Dark Aesthetic Theme
- User-Friendly Interface
- Clean Navigation
- Conversion-Focused Design

## Key Features

- Professional Homepage
- Membership Pricing Cards
- Virtual Trainer Support
- Fitness Equipment Section
- Free Trial Registration
- Contact Information
- Responsive Website Layout
