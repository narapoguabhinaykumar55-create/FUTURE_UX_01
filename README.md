# Lock In Fitness

## Future Interns - UI/UX Design Internship

### Task 1: Local Service Business Website Redesign

## Project Overview

Lock In Fitness is a modern fitness website UI/UX design created in Figma. The project focuses on user-friendly navigation, clean visuals, lead generation, and conversion-focused design to deliver a seamless user experience.

## Pages Designed

- Home Page
- Membership Page
- Services Page
- Contact / Free Trial Registration Page

## Tools Used

- Figma

## Features

- Modern and Responsive UI Design
- Membership Plans
- Virtual Trainer Support
- Fitness Services
- Free Trial Registration
- Contact & Lead Generation
- Clean Navigation
- Conversion-Focused Layout

## Design Goals

- Improve user experience
- Encourage membership registrations
- Increase lead generation
- Provide easy navigation
- Create a modern fitness brand identity
